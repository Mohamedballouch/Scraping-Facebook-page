# Scraping-Facebook-page
Scraping Facebook Hespress page

### Libraries 
I used Selenium and BeautifulSoup for scraping the posts.


# Example of scraping a Facebook Post of Hespress Page.

Output contains:
- Poste ID
- Date 
- Message
- Caption
- Link
- Type of the post
- Image og the post
- Number of Likes
- Number of Shares
- Number of Comments
